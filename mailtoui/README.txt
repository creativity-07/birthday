INTRODUCTION
------------

A simple way to enhance your mailto links with a convenient user interface.

REQUIREMENTS
------------

This module does not have any dependency on any other module.

INSTALLATION
------------

Install as normal (see http://drupal.org/documentation/install/modules-themes).

CONFIGURATION
-------------

This module does not have any configuration, only have to add 'mailtoui'
class for the mailto links
(eg: <a class="mailtoui" href="mailto:nesleepinto27@gmail.com">Contact</a>).
